## Features

- Create custom star types (name, meaning, colour, prize count)  
- Increase or decrease star counts with + / âˆ’ buttons  
- Automatic prize ticks appear when goals are reached  
- Inline editing for kids and star types  
- Reorder kids (move up / move down)  
- Reset all data with one button  
- Clean, responsive, child-friendly interface  
- Data saved automatically in localStorage  

---

## Usage

1. Download or clone the repo  
2. Open `index.html` in your browser  
3. Use the interface to add kids, create star types, and track rewards  
4. Click **Reset** (top right) to clear all stored progress
